🏥 Hospital Management System (MERN Stack)
Live App : https://user-interface-msm.netlify.app/

A full-stack Hospital Management System (HMS) built using the MERN stack to digitize hospital operations including patient management, appointment scheduling, medical records, and billing.

Designed to simulate real-world hospital workflows with secure role-based access control and scalable architecture.

🚀 Key Features

🔐 Role-Based Access Control (Admin, Doctor, Patient, Receptionist)

📅 Real-time Appointment Booking & Management

📁 Electronic Health Records (EHR)

💳 Automated Billing & Invoice Generation

🛡 Secure Authentication (JWT + Bcrypt)

📊 Admin Dashboard with Operational Insights

🛠 Tech Stack

Frontend: React.js, Axios, React Router
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Architecture: RESTful API (MVC Pattern)
